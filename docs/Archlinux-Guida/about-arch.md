# ¿Qué es Arch Linux?

Arch Linux es una distribución de Linux independiente desarrollada por la comunidad que prioriza la simplicidad, la modernidad, el pragmatismo, la centralización del usuario y la versatilidad. 

## Características principales

- **Ligereza**: Solo instala lo que necesitas
- **Rolling Release**: Siempre tienes la última versión de los paquetes
- **Pacman**: Gestor de paquetes moderno y rápido
- **AUR**: Arch User Repository - miles de paquetes contributed por la comunidad
- **Wiki excelso**: La documentación de Arch Linux es reconocida como una de las mejores

## Filosofía

Arch Linux sigue la filosofía KISS (Keep It Simple, Stupid) - mantén las cosas simples y Evita la complejidad innecesaria.

## ¿Por qué Arch Linux?

1. **Control total** sobre tu sistema
2. **Paquetes actualizados** constantemente
3. **Personalización** sin límites
4. **Gran comunidad** y documentación
5. **Aprendizaje** continuo

## Derivadas de Arch Linux

Algunas distribuciones basadas en Arch Linux:
- Manjaro
- EndeavourOS
- ArcoLinux
- ArchLabs
- BlackArch (seguridad)
