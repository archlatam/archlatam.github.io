# Derivadas de Arch Linux

Las derivadas (distribuciones basadas en Arch Linux) ofrecen diferentes enfoques y características adicionales.

## Principales Derivadas

### Manjaro
- Más popular derivada de Arch
- Instala drivers y software fácil
- Tres ediciones: GNOME, KDE, XFCE
- Herramienta de configuración gráfica

### EndeavourOS
- Instalación casi igual a Arch
- Entornos de escritorio variados
- Excelente documentación
- Comunidad activa

### ArcoLinux
- Totalmente personalizable
- Instaladores gráficos y de texto
- Escritorios: Openbox, i3, Qtile, BSPWM

### ArchLabs
- Minimalista y configurable
- Basada en Bspwm y Openbox
- Para usuarios intermedios y avanzados

### BlackArch
- Seguridad y auditoría
- +2900 herramientas de seguridad
- Para profesionales de ciberseguridad
