## [Arch Linux ARM (2009)](https://archlinuxarm.org/)

![image](https://archlinuxarm.org/forum/styles/simplecorp/imageset/ALARM-2.png)

Arch Linux ARM is a Linux distribution for ARM computers.

<br>

## [Archcraft (2021)](https://archcraft.io/)

![derivate](https://seeklogo.com/images/A/archcraft-logo-7C0C9A2F58-seeklogo.com.png)

Archcraft is a Linux distribution created on top of Arch Linux. It uses window manager and light applications, which makes it super fast.

<br>

## [Manjaro Linux (2011)](https://manjaro.org/)

![derivate](https://upload.wikimedia.org/wikipedia/commons/thumb/8/85/Manjaro_logo_text.svg/2560px-Manjaro_logo_text.svg.png)

Manjaro is a free and open source Linux operating system that emphasizes user privacy and hardware control.

<br>

## [ArcoLinux (2017)](https://arcolinux.info/)

![derivate](https://upload.wikimedia.org/wikipedia/commons/thumb/7/7d/Arcolinux.svg/598px-Arcolinux.svg.png?20180609082016)

ArcoLinux is a distribution based on Arch Linux. Development takes place in three branches - ArcoLinux, ArcoLinuxD and ArcoLinuxB.

<br>

## [Garuda Linux (2020)](https://garudalinux.org/)

![derivate](../images/garuda-purple.svg)

Garuda Linux is a Linux distribution based on Arch Linux.

<br>
