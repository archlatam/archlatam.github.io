The Arch Linux operating system also supports the Nano editing software distribution. Nano provides a lightweight and easy-to-use text editing environment on Arch Linux. Listed below are some of the commands usable in Nano on Arch Linux:

- `nano file_name`: Open an existing file in Nano
- `Ctrl + O`: Save the file
- `Ctrl + X`: Exit Nano
- `Ctrl + K`: Cut a line
- `Ctrl + U`: Paste a line
- `Ctrl + Space`: Insert cursor
- `Ctrl + W`: Search for a word in the text
- `Ctrl + G`: Display Nano help
- `Ctrl + C`: Display information about selected text

To save a file and change the save name, you can use the following command:

- `:w file_name` Save the file with a new name

This way, the file will be saved with the new name specified after the colon and "w". Nano on Arch Linux continues to be a reliable and simple solution for text editing.
